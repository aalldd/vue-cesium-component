# 卷帘分析

municipal-rolling 卷帘分析组件，用于将地上模型通过拉窗帘的方式，卷开，可以更加直观的看到管线与地上模型的对应关系

# 效果一览

# 基本用法

基本的卷帘分析用法,用户可以去自己指定卷帘线与地形的偏移值

<template>
  <municipal-roll :offset="offset">
  </municipal-roll>
</template>

<script>
export default {
  name: "RollingAna",
  data() {
    return {
      offset: [0, 0]
    };
  }
};
</script>
